What is favicon?

A favicon is the small icon that your user sees on thier browser tab when they visit your website. The icon we created in this lab is an example. It is also knowns as a shortcut icon, website icon, tab icon, URL icon, or bookmark icon. It is a file containing one or more small icons associated with a particular website or web page.

How are favicons created?

  1. Create it in a graphical editor like Photoshop, GIMP, or PicMonkey.
  2. Draw it up in special online services, such as favicon.cc or studio.code.org and then download it from there in a suitable format.
  3. Them, use an existing image of, for example, your logo.

How did I create a favicon for this site?

In studio.code.org, I seth the image width and height to 27 pixels each and 12 bits per pixels to 12. In binary, I messed around with the numbers that resulted in a logo.

What does your icon look like, in binary and/or hexadecimal?

My icon has a gray background along with my initials in blue with a diamond shape surrounding it.
